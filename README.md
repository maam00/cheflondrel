# ChefLon

Private pizza experiences by Chef Lonny Geller — NYC, LA & beyond.

**Live site:** [cheflon.com](https://cheflon.com)

## About

ChefLon is a single-page website for Chef Lonny Geller's private pizza party catering business. Omakase-style wood-fired pizza experiences for parties, weddings, and events.

## Tech

- Single `index.html` — no build step, no framework
- Hosted on GitHub Pages with custom domain
- Calendly embed for booking
- Google Analytics 4
- Fully responsive, CSS animations, lazy-loaded images

## Setup

1. Clone the repo
2. Open `index.html` in a browser — that's it

For deployment, push to `master` and GitHub Pages handles the rest.

## Domain

Custom domain `cheflon.com` configured via:
- `CNAME` file pointing to `cheflon.com`
- Namecheap DNS: A records → GitHub Pages IPs, `www` CNAME → `maam00.github.io`
